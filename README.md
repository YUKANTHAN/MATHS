# MATHS

A collection of Python scripts designed to solve various mathematical equations and problems efficiently.

## Features
- Solves complex mathematical equations.
- Lightweight and easy to use.

## How to Run
1. Ensure you have Python installed (Python 3.x recommended).
2. Clone this repository: `git clone https://github.com/YUKANTHAN/MATHS.git`
3. Run the scripts using: `python <script_name>.py`
